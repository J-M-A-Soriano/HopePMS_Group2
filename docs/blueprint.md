# **App Name**: ProdSync

## Core Features:

- User Authentication: Secure user registration, login, logout, and session management leveraging Supabase Auth.
- Product Listing View: A dashboard displaying a comprehensive list of all managed products, with basic search and filter capabilities.
- Product Detail & Management: Pages for viewing detailed product information, and forms to add new products or update existing product details through Supabase.
- Admin Product Control Panel: A dedicated section for administrators to oversee and manage all product data, including more sensitive actions like bulk updates or deletion.
- Role-Based Access Control (RBAC): Enforce permissions to restrict access to certain application features and data based on assigned user roles (e.g., admin, standard user) utilizing Supabase Row Level Security.
- AI Product Description Tool: A generative AI tool to assist in drafting product descriptions based on provided keywords, product category, or brief input from the user.

## Style Guidelines:

- Primary color: A rich, deep indigo-blue (#3929A3) suggesting stability, professionalism, and digital precision.
- Background color: A very light, desaturated version of the primary's hue (#F7F5FC), providing a clean, spacious canvas for data and forms.
- Accent color: A clear, vibrant blue (#3399FF), used for interactive elements like buttons, links, and highlights, offering a refreshing contrast while remaining harmonious.
- Primary font: 'Inter' (sans-serif) for all text elements, chosen for its modern, neutral, and highly readable qualities, ensuring clarity across data tables and descriptions.
- Utilize professional, streamlined vector icons from Lucide React that clearly represent actions and data types without cluttering the interface.
- Employ a structured, grid-based layout with a consistent sidebar navigation and a clear main content area, optimized for data density and ease of scanning.
- Incorporate subtle and swift animations for state changes, feedback, and transitions, enhancing user experience without causing delays.
